# Disability-Health-Behaviors
An analysis of the health behaviors of disabled people.
See the live dashboard here: https://public.tableau.com/app/profile/prudence.theodorakos/viz/DisabilityHealthBehaviors/Story1

Original dataset not avalible but is based on the 2022 Behavioral Risk Factor Surveillance System, which can be found here https://www.cdc.gov/brfss/annual_data/annual_2022.html
